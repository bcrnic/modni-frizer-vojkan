# Modni Frizer VOJKAN — Web sajt

Ovo je statički/SPA web sajt za frizerski salon **Modni Frizer VOJKAN**.

Sajt sadrži:

- početnu (Hero) sekciju
- informacije o salonu
- listu usluga
- galeriju radova
- kontakt sekciju

## Tehnologije

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui (Radix UI komponente)
- framer-motion

## Lokalno pokretanje

Preduslov: Node.js (preporuka: Node 18+)

```bash
npm install
npm run dev
```

Aplikacija će biti dostupna na:

- http://localhost:8080

## Build

```bash
npm run build
```

Build izlaz je u folderu `dist/`.

## Skripte

- `npm run dev` — development server
- `npm run build` — production build
- `npm run preview` — preview buildu-ovanog sajta
- `npm run lint` — ESLint
- `npm test` — Vitest
