# Modni Frizer VOJKAN

This repository contains a single-page website for the hair salon **Modni Frizer VOJKAN**.

## Tech stack

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui (Radix UI)
- framer-motion

## Running locally

Requirements:

- Node.js 18+

Install dependencies and start the dev server:

```bash
npm install
npm run dev
```

The app will be available at:

- http://localhost:8080

## Build

```bash
npm run build
```

Production output is generated in `dist/`.
